# Mijn kookboek

Dit is een voorbeeld implementatie van hoe een kookboek gemaakt kan worden met React.

Check het project uit door middel van `git clone git@github.com:joppe/techgrounds-webdev.git`.
Voer daarna in de root van het project `npm install` uit.
Het project kan gestart worden door middel van `npm run dev`. Open een browser tab en ga dan naar `http://localhost:3000`.

Een demo van het project is te zien op [techgrounds-webdev.vercel.app](https://techgrounds-webdev.vercel.app/).
